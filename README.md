### About

An elegant screenshot curation tool combining sophisticated design with powerful workflow features. No installation required - works directly in your browser!

### âœ¨ Features

- **6 Customizable Categories**: Personalize category names to fit your needs
- **Elegant Design**: Serif typography (Playfair Display) with warm color palette
- **One-Click Zoom**: Click any image to view full-screen
- **Chrome Bookmark Bar Style UI**: All categories visible in one horizontal bar

- **Favorites Filter**: View only starred screenshots
- **Multiple Sort Options**:

  - Oldest First (oldest to newest)
  - By Date (grouped by date with section headers)
- **Personal Branding**: Customize archive title and curator name
- **Backup & Restore**: Export/import as JSON
- **Export Options**:

  - Export as PDF (with notes and dates)
- **Notes**: Add notes to each screenshot
- **Auto-Advance**: Automatically move to next after archive/delete
- **Privacy-Focused**: All data stored locally
